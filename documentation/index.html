Introduction

SMS Management Application (SMA) is an API based application used for managing sms among users.

Contacts

Add Contacts

POST https://us-central1-smsmangement.cloudfunctions.net/sms/contacts

Creates a new contact

Headers
Content-type application/x-www-form-urlencoded
Key Value
name james
phoneNumber 1234567 
{
    "response": "Contact added successfully"
}

Get Contacts

GET https://us-central1-smsmangement.cloudfunctions.net/sms/contacts

GETS all contacts 

{
    "response": [
        {
            "name": "james",
            "phoneNumber": "1882828282",
            "key": "-LdJnXBv4tne_GL2uOJ8"
        }
    ]
}

Delete a contact

DELETE https://us-central1-smsmangement.cloudfunctions.net/sms/contacts/:key/:phoneNumber

Deletes a contact

Parameters
phoneNumber 1882828282
key -LdJnXBv4tne_GL2uOJ8 
{
    "response": "Contact deleted"
}

SMS

Send SMS

POST https://us-central1-smsmangement.cloudfunctions.net/sms/send-sms

Send a new message

Headers
Content-type application/x-www-form-urlencoded
Key Value
senderNumber 48937834
phoneNumber 1234567
name user
sms hello world 
{
    "response": "Message sent successfully"
}

Get received SMS

GET https://us-central1-smsmangement.cloudfunctions.net/sms/get-sms/:phoneNumber

GETS received SMS for a contact

Parameters
phoneNumber 1234567 
{
    "response": [
        {
            "name": "user",
            "phoneNumber": "1234567",
            "senderNumber": "48937834",
            "sms": "hello world"
        }
    ]
}

Get sent SMS

GET https://us-central1-smsmangement.cloudfunctions.net/sms/get-sent-sms/:phoneNumber

GETS sent SMS for a contact

Parameters
phoneNumber 48937834 
{
    "response": [
        {
            "name": "user",
            "phoneNumber": "1234567",
            "senderNumber": "48937834",
            "sms": "hello world"
        }
    ]
}
